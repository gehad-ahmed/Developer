# 💼 Developer — Personal Portfolio Template

A responsive personal / creative portfolio website with sections for services, portfolio, team and contact, built with HTML, CSS and JavaScript.

![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)

## 📖 Overview

A clean, modern one-page portfolio/agency site with a hero, about, services, portfolio and contact sections — fully responsive and easy to customize.

## ✨ Features

- 🎯 Hero / intro section
- 🛠️ Services and portfolio showcase
- 👥 Team / about section
- ✉️ Contact section
- 📱 Fully responsive layout

## 🛠️ Tech Stack

- HTML5 & CSS3
- Vanilla JavaScript (ES6+)

## 🚀 Getting Started

```bash
git clone https://github.com/gehad-ahmed/Developer.git
cd Developer
```

Then open `index.html` in your browser (or use the VS Code **Live Server** extension).
